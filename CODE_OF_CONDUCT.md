# Code of Conduct

We expect all contributors, maintainers, and users to communicate professionally and respectfully.

## Standards

- Be constructive
- Be direct without being hostile
- Assume good intent
- Keep feedback focused on the work, not the person

## Enforcement

If behavior is abusive, harassing, or disruptive, maintainers may remove comments, close issues, or restrict participation as needed.

## Reporting

Use the security or issue channels in this repository if you need help raising a concern.
